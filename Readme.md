# Para utilizar ele

### Rails 7, ruby 3.1.3, nodejs v18 

# BACKEND

## Entrar no backend

```cd backend```

## instalar as gems

```bundle install```

## criar o banco de dados

```rails db:create db:migrate db:seed```

## subir o servidor

```rails s -p 3001```

# FRONTEND

## entrar no frontend

```cd frontend```

## instalar as dependencias

```npm install```

## iniciar o frontend

```npm start```
